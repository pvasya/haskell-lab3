# haskell-lab3

Створити програму обробки тексту підручника з програмування (який знаходиться у файлі) на Haskell з використанням типів: Символ, Слово, Пропозиція, Розділовий знак та ін. У всіх завданнях з формуванням тексту замінювати табуляції та послідовності пробілів одним пробілом.

5. У кожному реченні тексту поміняти місцями перше слово з останнім, не змінюючи довжини речення.
